# Workers

Фоновые задачи будут жить отдельно от request/response кода:

- уведомления;
- payment webhooks follow-up;
- пересчет внутренних балансов;
- сверки выплат;
- stale order checks;
- будущая автоматизация назначения курьеров.

На старте worker может быть отдельным npm-скриптом в этом же репозитории.
